Простой SSH клиент
* возможность сохранить ранее введенные подключения
* возможность сохранить ранее введенные команды
* кастомные команды
* возможность скачивать файлы по известному пути
* пароли хранятся зашифровано в отдельном файле, ключ шифрования изменен в скомпилированной версии
* можно вписывать отдельные профили с незашифрованным паролем в save.json
* ранее введенные команды храняться в отдельном файле списке - history.txt
* не требует установки и зависимостей, код на питоне, исходник открыт
